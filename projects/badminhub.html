Project Overview

BadminHub is a mobile application designed to assist badminton event administrators with player attendance management, player pairing (for double matches), and financial flow management. It streamlines event handling, making it easier for admins to organize badminton events and ensure smooth operations.

Technologies Used

  • React Native: Front-end framework for building mobile applications.
  • NodeJS: Back-end JavaScript runtime for handling server-side logic.
  • MySQL: Database system for managing player and event data.

Project Setup

To set up the project locally, follow these steps:

  1. Clone the repository: 
    git clone <repository-url>
  2. Install dependencies: 
    npx expo install
  3. Run development build: 
    npx expo run:android

Key Features

  • Player Attendance Management: Track player attendance and manage status changes.
  • Player Pairing: Automatically pair players for double matches based on skill level.
  • Financial Management: Track payments, expenses, and financial flow during events.
